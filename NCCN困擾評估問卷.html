⚙️ Google Sheets Apps Script 設定說明
1開啟 Google Sheets → 上方「擴充功能」→「Apps Script」
2全選刪除原有程式碼，貼上以下內容後儲存： 
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(["填寫時間","困擾分數","困擾程度","勾選問題項目","建議轉介資源"]);
  }
  var p = e.parameter;
  sheet.appendRow([p.t, p.s, p.l, p.q, p.r]);
  return ContentService.createTextOutput("ok");
}
3點「部署」→「管理部署作業」→ 鉛筆圖示 → 版本選「建立新版本」→「部署」
！ 網址不會改變，不需要重新貼網址。部署完成後填寫問卷即可自動寫入試算表。
